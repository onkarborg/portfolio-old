Head Up!

Developed in Unity3D for Android phone, this application is mainly designed to work with Google Cardboard. Having Minigames which work mainly on the user keeping their heads stationary, this application will train kids to be stationary during MRI scans

Ducklings

This Application makes the documentation of your kids memories, may they be photos, videos, or notes, really quick and simple, with options to let collaborators contribute to the albums or documents.

Campus Nosh

This app is for providing easy and a variety of food items and for on campus students. Tracking food habits and prompting suggestions as well as pre-ordering food to reduce wait time are some of the main features of this application

Google Inbox

Analysing User interaction while using Google Inbox. Users were asked to perform some simple and unique tasks which proved to be a challange to some. Overall System Usability came out to 45%.

Intelligent Surface

An IDE developed using Microsoft Visual Studio and OpenFrameworks specifically to design and develop applications to run on TouchTables. A Composer and A Player would be used to develop and run the applications respectively. All the interactions included in the developed application could be set without writing any code at all

Mimicry Engine

A back end engine to match your action against the saved actions. Using Full Body Tracking and the Microsoft Kinect. Tthe first step would be to train the system for as many action as needed, then using an algorithm designed from scratch, it would match user actions with the saved actions and return a percentage similarity.
